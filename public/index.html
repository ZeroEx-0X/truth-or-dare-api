Truth or Dare API

Welcome to the ultimate Truth or Dare API! Get random truths or dares in multiple languages.

API Usage

Use the following endpoint to get a random Truth or Dare:

GET /TorD?type=T&lang=bn

Parameters:

  • type: T (Truth) or D (Dare)
  • lang: bn (Bengali), en (English)

Example Response:

{
  "type": "Truth",
  "content": "What is the most embarrassing thing you've ever done?"
}
